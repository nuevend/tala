How the System Works

The Vehicle Fuel Monitoring System is designed to help users track, analyze, and improve their fuel efficiency through simple inputs and automated calculations.

To begin, users enter the distance traveled in kilometers and the fuel consumed in liters. After selecting the appropriate vehicle type, the system calculates fuel efficiency by dividing distance by fuel usage. This provides a clear measurement of how efficiently the vehicle is performing.

The Add Record button stores the current data entry, allowing users to build a history of their trips. This recorded data becomes the foundation for further analysis.

The system also uses predefined reference values for different vehicle types to provide meaningful comparisons. These values represent the average fuel efficiency (km per liter) for each type of vehicle under normal driving conditions.

The system is based on the following reference data:

  • Motorcycle – 35 km/L (highly fuel-efficient)
  • Sedan – 15 km/L (balanced daily use)
  • SUV – 10 km/L (higher fuel consumption)
  • Truck – 6 km/L (heavy-duty, lowest efficiency)
  • Jeep – 8 km/L (moderate efficiency)

These values are based on general real-world averages commonly observed across vehicle categories. They are not exact for every model, but they provide a reliable baseline for comparison.

By comparing the user's actual fuel efficiency with these reference values, the system can determine whether the vehicle is performing efficiently, below average, or wasting fuel.

The History function displays all saved entries, helping users review past trips and observe patterns in their fuel consumption over time.

The Summary feature provides an overview of the user’s fuel performance. It calculates averages and highlights general efficiency trends, making it easier to understand overall vehicle performance.

The Trend function analyzes changes in fuel usage across multiple records. It helps users determine whether their efficiency is improving, declining, or remaining consistent.

Lastly, the Clear All button removes all stored data, allowing users to reset the system and start fresh.
